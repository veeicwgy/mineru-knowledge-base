## Stable Facts
- MinerU is a document parsing platform from OpenDataLab.
- Inputs include PDF, images, Word, PPT, and HTML.
- Outputs include Markdown, JSON, LaTeX, and HTML.
- MinerU has open-source deployment, online API, desktop client, CLI/SDK, MCP, and RAG ecosystem integrations.

## Current API Notes
- Precision API: token required, live docs currently show <= 200MB and <= 200 pages.
- Agent lightweight API: no token, live docs currently show <= 10MB and <= 20 pages.
- Historical materials may mention 600 pages or 2000 daily high-priority pages; verify live docs before publishing.
- Licensing statements may also drift across sources; verify the main MinerU repository README and LICENSE before publishing current license claims.
